Setup Instructions

Migrating Existing Signatures

If you have existing signatures, then the data will need to be migrated. Data has already been migrated for

  • Spring Team - use Contributor Agreement "spring"

Linking to Your Repository

The next step is to link your project to a contributor license agreement. Navigate to Link to CLA.

  • Select the repositories you wish to link to a CLA
  • Select the Contributor Agreement you wish to associate to your project.

Additional Steps

Afterwards you will see a success message along with some additional steps. Follow the instructions provided to:

  • Update CONTRIBUTING
  • Update Existing Pull Requests

User Instructions

Creating a Pull Request

The first time users create a pull request they will be instructed to sign the Contributor License Agreement

Selecting a CLA to Sign

Users are then instructed to sign the CLA. Most users will select to sign the Individual Contributor License Agreement.

Signing Individual Contributor License Agreement

The user can then sign the Individual Contributor License Agreement. After signing, the UI will then prompt the user to return back to the Pull Request.

Updated Pull Request

The Pull Request indicates that the Pull Request has been successfully signed.

Already Signed

The next Pull Request will indicate that the user has already signed the agreement.
